# Update 1.4.5 - Seasonal Wanderers

## Changes
- You can now find Seasonal Wanderers occasionally. They will give you certain quest items in exchange for Anima Ingots.
- Some quest items have been added to Archaeology loot tables.
- Increased range of Divine Smite from 3.5 to 5.
- Increased casting time of Divine Smite from 0.5 seconds to 1.25 seconds.
- Divine Smite now has a new animation.
- New model and texture for Reaper.
- Aggravate, in the Oath of Vengeance Spellbook, now works as intended.
- The Avatar of Malzith animations have been updated.
- Talent Trees for all relics are now complete.

## UI Changes
- Seasonal tooltips now display the correct translation for Velmoria's seasons.
- Spell Power attribute icons now correctly display on all tooltips.
- Updated many ingot icons to fit in with the style of Resurgence.
- Updated lang files for Cosmetic Overrides.
- Updated Memory of Aethara dimension title.

## Slimes SMP Changes
- Added Enchanting guide in the quest book.
